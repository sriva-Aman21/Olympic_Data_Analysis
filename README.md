# 🏅 Olympics Analysis Web App

An interactive **Streamlit web application** for analyzing historical Olympic Games data.  
This project provides insights into medal tallies, country performance, athlete statistics, and trends over time.

---

## 🚀 Features

### 📊 Medal Tally
- Country-wise medal count
- Year-wise filtering
- Historical performance comparison

### 📈 Overall Analysis
- Number of Olympic editions
- Participating nations over time
- Events and sports growth trends
- Most successful athletes

### 🌍 Country-wise Analysis
- Medal trends over years
- Sport-wise performance heatmap
- Top athletes per country

### 👥 Athlete-wise Analysis
- Age distribution of athletes
- Height vs Weight analysis
- Gender participation trends

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Pandas & NumPy**
- **Matplotlib & Seaborn**
- **Plotly**
